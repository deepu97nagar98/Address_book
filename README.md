# Flask Address Book

###  Features
- Modular design
- In-memory data storage
- Follows API spec from the test

###  How to Run

```bash
git clone <this-project>
cd addressbook_flask
python3 -m venv venv
source venv/bin/activate
pip install Flask
python app.py
```

Server starts at: `http://127.0.0.1:5000`
